# TaLL Conf Landing Page

A modern, responsive conference landing page built with **HTML** and **CSS**. This project recreates a clean event website featuring a hero section, conference sessions, event details, and a polished purple-themed design.




 # Features
- Hero section with call-to-action button
- Navigation bar
- About section
- Conference session highlights
- Circular speaker/session images
- Event details cards
- Modern typography and spacing
- Purple-themed user interface

# Built With

- HTML5
- CSS3
- Flexbox

# Inspiration
This project was built as a front-end practice to improve HTML and CSS skills.
